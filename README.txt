RimWorld Vietnam
================

This is the official Vietnam language translation data for RimWorld alpha 15.

See this page for license info:
http://ludeon.com/forums/index.php?topic=2933.0

Official Translators:
---------------------
- Hoàng Lê

How to install this language pack:
1. Extract to <RimWorld path>\Mods\Core|Languages
2. Rename folder "RimWorld_Vietnamese_Official_Final" to "Vietnam"
3. Start RimWorld, click Options, click Choose Language, select Vietnamese.. Done

Any suggestion about grammar, error, ect.. please comment in this thread
https://ludeon.com/forums/index.php?topic=26500.0
